定义变量的时候尽量赋初始值

## 从解决方案到计算机程序（结构化编程）

1. 面对一个问题，你必须要先找到解决这个问题的方法，然后才有可能写出相应的程序。还没有想到解决方案的时候，不要急着动手去写程序
2. 先粗后细、先抽象后具体
3. 写程序前，先构想好程序的结构
4. 先写程序轮廓，而后再补充变量定义等细节

找到解决方案 -> 描述成程序 -> 编写代码
